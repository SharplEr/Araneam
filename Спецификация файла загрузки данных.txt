Описания необходимого формата файла для использования для загрузки данных из других файлов. Используется в методах класса DataFile, и потом оттуда везде где только можно. Расписано по строкам:

1. Имена файлов, из которых будут браться данные, по имени на одну строку. Должен быть хотя бы один.
2. Пустая строка разделитель (можно заполнить пробелами -- не важно).
3. Имена полей рассматриваемых как входные значения, по имени на одну строку. Должен быть хотя бы один.
4. Пустая строка разделитель.
5. Имена полей содержащие непрерывные значения, по имени на одну строку. Может не быть ни одного. Если ни одного нет, то пустых строк будет две (под 4 пунктом и под 6).
6. Пустая строка разделитель.
7. Имя поля рассматриваемое как выходное значение. Одна обязательная строка.
8. Пустая строка разделитель.
9. Либо конец файла, либо строчка "standard" (без кавычек), либо имя файла dll библиотеке. Если конец файла или строчка "standard", то используется обычный Convert.ToDouble для преобразование из строки в вещественное число для полей помеченных как непрерывные. Любая другая строчка воспринимается как имя на сборку, в которой находится класс наследующий IToDouble, оттуда берется статический метод ToDouble и используется для преобразования.
10. Можно указать имя класса из которого возьмется метод ToDouble из файла указанного в 9. Класс все равно должен наследовать IToDouble на всякий случай.
11. Любой текст идущий далее не будет восприниматься кодом. Можно оставлять комментарии.
